### 좌측 사이드바

> 최종본에서 **5개 항목이 같은 depth의 형제**로 정리됨: 연동 인스타 계정 / 콘텐츠 관리 / 자동화 설정 / 콘텐츠 성과 / 프로필.

- **0.2.3 연동 인스타 계정**
  - **AI 기획 리스트** — 테이블 형식으로 기획·제작
    1. No.
    2. 형식 (카드뉴스 / 사진 첨부형 카드뉴스)
    3. 카드뉴스 장 수 선택
    4. 타이틀(주제)
    5. 서브 타이틀 + 본문 내용 × 장수
    6. 제작하러 가기 버튼
    - 기획 추가 가능
    - **AI 제작 및 편집**
      - 카드뉴스 제작
      - 템플릿 둘러보기
    - **최종 검수 및 업로드** 📱 — AI 초안 생성(템플릿 + 브랜드컬러)
      - `[명시]` 'AI 생성물' 명시하되, 사용자가 편집을 거치면 해제
      - `[체크]` 출처 확인 · 민감 표현(권유·강요) 점검 · 표기 누락 체크
      - `[사용자 승인]` 업로드 전 반드시 사용자에게 승인/거절 입력 받기 **(신뢰 확보 핵심 단계)**
      - *AI 저작권 법령 참고: korea.kr 자료(newsId=148927815)*

- **0.2.4 콘텐츠 관리**
  - **내 콘텐츠 히스토리(칸반보드)** — 칸반보드 형식
    - 가로 열 **6개**: 기획 중 / 기획 완료 / 제작 중 / 제작 완료 / 예약 업로드 / 업로드 완료
    - 세로 행 스크롤: 세로 1000px 영역 고정 후 안에서 가로 열별 스크롤
  - **캘린더 / 리스트(토글)로 콘텐츠 관리**
    - 관련 게시물 / 임시저장 토글

- **0.2.5 자동화 설정 → DM 리드마그넷** — 키워드 트리거 + 보낼 자료(정리본 링크) 등록
  - 댓글(갯수는 요금제 기준)에 따라 DM 발송
  - **요금제 기준**: 베이직(무료회원) 100건 / 프로 1,000건 / 프리미엄 제한 없음

- **0.2.6 콘텐츠 성과**
  - **인스타 인사이트**
    1. **계정 단위로 가져오는 정보**
       - 현재 총 팔로워 수(`followers_count`) — 호출 시점 값
       - 팔로워 유입/이탈 분리(`follows_and_unfollows`) — [예외처리] 100명 이상부터 제공
    2. **게시물 단위로 가져오는 정보** (게시물 발행 이후 누적값)
       - 조회(`views`), 도달(`reach`)
       - 저장(`saved`), 공유(`shares`), 좋아요/댓글(`likes`/`comments`)
       - 프로필 방문(`profile_visites`), 게시물 기여 팔로우(`follows`)
    3. **DM 리드마그넷 데이터** — 자체 카운트
    - 인사이트 업데이트 날짜 표기(`YYYY.MM.DD 04:00:00`) — 오전 4시 이유: 수치 안정 시간대 + 서버/API 요청 부담 분산
    - 새로고침 버튼 · [참고] Meta 공식문서
  - **팔로워 달성 챌린지 / 업로드 릴레이 (기여 그래프)** — 매일/매주 업로드를 칸으로 쌓아 업로드에 대한 꾸준함을 시각화로 표현 (GitHub 잔디 형식)
    - 1칸 = 하루 / 칸의 농도 = 그날 발행 확정 건수 (0·1·2·3·4+ : 색 5단계)
    - **축 라벨(사용자 이해용·필수)**: 상단 가로축 = 월(月) 라벨 / 좌측 세로축 = 요일 라벨. [의도] 칸의 "언제"를 라벨만 보고 바로 읽게 한다(GitHub 기여 그래프와 동일한 읽기 규칙)
    - **발행 확정 카운트 규칙**: 발행 성공 시 자동 카운트 / 예약 발행은 실제 발행된 날에 칸 적립
    - **마우스 호버 / 클릭 시 [툴팁] 노출**: 해당 칸의 `YYYY년 M월 D일` + `업로드 N건`을 수치로 표시(0건인 날도 "0건" 명시). [의도] 색 농도만으론 모호한 "정확히 며칠·몇 건"을 보완
