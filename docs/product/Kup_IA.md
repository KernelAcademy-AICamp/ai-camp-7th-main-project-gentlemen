### 좌측 사이드바

- **0.2.3 콘텐츠 인스타 계정**
  - 테이블 형식으로 기획·제작됨. **AI 기획 리스트:**
    1. No.
    2. 형식 (카드뉴스 / 사진 정사각형 카드뉴스)
    3. 카드뉴스 장수 선택
    4. 타이틀(주제)
    5. 본문 타이틀 + 본문 내용 × 장수
    6. 제작하러 가기 버튼
    - 기획 추가 가능

  - **AI 제작 및 편집**
    - **0.2.3.2 AI 기획 리스트** — 카드뉴스 제작 / 템플릿 둘러보기

        - **0.2.3.3.2**

  - **0.2.3.4 콘텐츠 관리** — 내 콘텐츠 대시보드
    - 칸반보드 형식 — 가로 4단계(기획 중 / 기획 완료 / 제작 중 / 제작 완료)에서 카드(게시물) 관리
    - 셰도우/스크롤 형식, 사진 1000px 무빙 고정 후 가로 길이별 스크롤 가능 방식(?)

    - **0.2.3.4.2**

  - **0.2.3.4 자동화 설정** — DM 리드마그넷
    - 카드 트리거 + 보낼 자료(정리값 등록), 댓글 → 요금제 기준에 따라 다르게 DM 발송
    - 요금제 기준: 베이직 × 100개 / 프로 × 1000개 / 프리미엄 × 제한 없음

  - **0.2.3.5 콘텐츠 성과** — 인스타 인사이트
    1. 계정 단위로 가져오는 정보 — 현재 총 팔로워 수(`followers_count`, 오늘 시점값) / 팔로워 유입·이탈(`follows_and_unfollows`)
    2. 게시물 단위로 가져오는 정보 — 게시물별 누적값: 도달(`reach`) / 저장(`saved`) / 공유(`shares`) / 좋아요(`likes`) / 댓글(`comments`) → 프로필 방문(`profile_visits`) / 팔로우(`follows`) *[참고: Meta 공식 문서]*
    3. DM 리드마그넷 데이터 자체 카드 — 인사이트 업데이트 날짜 표시(YYYY.MM.DD 04:00:00) / 설정 이후 수치(데이터 안정적인 시간대, 비교값)(?)
    - **0.2.3.5.1 월별 달성 챌린지** — 매일/매주 업로드 카드로 봐야 할 업로드에 대한 꾸준함을 시각화로 표현. 1주차+ / PV 노출 → 그날 발행 횟수(0,1,2,3,4+ 색 5단계). 발행 성공 시 자동 카운트, 발행 실패도 날짜에 표기(?)
